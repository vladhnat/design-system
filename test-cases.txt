---------------------------------------------------
Goal: Verify the main page container and layout structure are rendered correctly with proper responsive behavior

Steps:
- Navigate to the home page
- Inspect the page structure and verify:
  * Page has a full-width container with zinc-50 background (light mode) or black background (dark mode)
  * Main content area is centered with max-width of 3xl
  * Content has proper padding (py-32 px-16)
  * Layout is responsive (flex-col on mobile, adjusts on larger screens)
- Check browser tab title and page metadata
- Assert that page title is "Create Next App"
- Assert that page description is "Generated by create next app"
